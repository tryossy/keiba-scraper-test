# 競馬データ自動スクレイピングツール

GitHub Actions を使用して、自動的に競馬データを取得するツールです。

## 概要

netkeiba.com から競馬レースデータを自動取得し、Google Drive に保存します。

**主な特徴:**
- GitHub Actions による自動実行（Public リポジトリで無制限利用可能）
- レート制限対応（サーバー負荷軽減）
- Google Drive への自動バックアップ
- 月次・週次・年次の柔軟な実行設定
